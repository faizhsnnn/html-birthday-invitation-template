# HTML Birthday Invitation Template

## Overview

This project is a simple HTML-based birthday invitation template designed to practice semantic structure, content hierarchy, media embedding, and external linking using standard HTML.

Although visually minimal, the focus of this project is correctness and clarity—ensuring that content is well-structured, readable, and maintainable using semantic HTML elements.

This project was built as part of **#90DaysOfCode**, with the intent of strengthening frontend fundamentals that support scalable frontend and full-stack development.

---

## Key Concepts Demonstrated

- Proper HTML document structure
- Semantic use of headings and lists
- Image embedding with accessibility attributes
- External linking for location references
- Clean and readable markup
- Separation of content structure from styling
